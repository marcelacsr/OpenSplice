Detailed Description

The proprietary builtin CMDataReaderBuiltinTopicData topic.

This topic is normally only used by OpenSplice tools.

Note
Users are discouraged from doing anything with this topic, so as not to interfere with internal mechanisms that rely on them. The structure of these topics may change without notification.
